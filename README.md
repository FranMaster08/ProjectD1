# Proyecto 

Esta en una app para que te relajes y puedas pensar Mejor.

### Instalación 🔧

_Primero debes instalar las dependecias del proyecto, esto lo lograras con esta instruccion_

```
npm install
```

_Para iniciar el servidor por favor usar_

```
npm start
```
